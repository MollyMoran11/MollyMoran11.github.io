Strategy #1

In multiplication we can use the picture strategy. For example, if the problem states 5x4 we can draw a picture where there are five groups and four dots in each to count up that this would be 20 total. You can also reverse this by having four groups and 5 dots in each. Be careful with this strategy because if the dots are unclear you could miscount the total.
